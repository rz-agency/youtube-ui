# YouTube UI Clone

A simple, responsive YouTube-like frontend interface built to practice CSS layout techniques.

## Features
* **Header:** Search bar, logo, and user icons.
* **Sidebar:** Collapsible navigation menu.
* **Video Grid:** A responsive layout using CSS Grid and Flexbox.
* **Responsive:** Designed to work on different screen sizes.

## Technologies Used
* HTML5
* CSS3 (Grid & Flexbox)

## How to Use
Simply download the files and open `index.html` in your browser to see the UI in action.
